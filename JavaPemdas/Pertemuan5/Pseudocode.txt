hour, minute, hourToSecond, minuteToSecond, second : integer

Start
Input hour
Input minute

hourToSecond <-- hour * 3600

num1, num2, num3, max, min, mean : float

Start
Input num1
Input num2
Input num3

mean <-- (num1 + num2 + num3) / 3
